# Point-Voxel Convolution CUDA Extensions

This module contains CUDA implementations for operations from
[PVCNN: Point-Voxel CNN for Efficient 3D Deep Learning](https://arxiv.org/abs/1907.03739).

They were taken from the [official implementation](https://github.com/mit-han-lab/pvcnn/tree/master/modules) and
modified to be installed as a Python package.
